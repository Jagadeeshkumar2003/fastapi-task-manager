 # A Full-Stack Multi User Task Application
 
A full stack application built with FastAPI on the backend and HTML,CSS,JavaScript on the frontend. It demonstrates user authentication, database management, and CRUD (Create, Read, Update, Delete) operations, all running on a clean, containerized MySQL database.

# Features
1. User Authentication: Secure sign-up and login system using JSON Web Tokens
2. Task Management: Create, view, update, and delete personal to-do items
3. Multi User Data: Each user's data is completely separate and secure
4. Interactive API: Automatic Swagger UI documentation for testing endpoints
5. Secure Password Hashing: Uses the modern Argon2 algorithm
6. Clean Architecture: Uses Pydantic for validation and SQLAlchemy for database interaction
7. Dockerized Database: MySQL database runs in an isolated Docker container for an easy setup
